PyLinden

PyLinden是Python实现的一个静态博客生成系统。

马上获取(Github) 查看文档

2013-11-08:持续重构ing。

此站点便是PyLinden生成的博客实例。

特征:

  • 静态页面，没有数据库，没有动态请求，没有大Django。
  • 支持部署到BAE（Baidu App Engine）。
  • 纯文件存储，一切皆文件。
  • 支持代码着色（pygments）。
  • Github Pages兼容，几乎只是Copy过来的事情。
  • 增量生成，逻辑上没有修改则不会产生写磁盘操作。
  • 支持（且仅支持）markdown格式书写日志。
  • 基于简单够用的设计哲学构建。

感谢:

  • 前端：Jquery, Bootstrap-v3
  • Python：pyYAML, markdown, jinja2, pygments, web.py
  • 部署：BAE(Baidu App Engine)
